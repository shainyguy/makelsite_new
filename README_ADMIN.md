# Инструкция по управлению каталогом MAKEL

## Основные изменения

### 1. Структура каталога
- **Серии** — дизайнерские линейки (Lillium, Mimoza, Defne...)
- **Товары** — конкретные изделия (выключатели, розетки)
- **Цвета** — доступные цвета для всех товаров

### 2. Управление в админке

#### Раздел "Серии"
- Добавление новых дизайнерских линеек
- Загрузка обложек серий
- Управление отображением на сайте

#### Раздел "Товары"
- Добавление товаров в серии
- Загрузка фото по цветам
- Указание цен для каждого цвета

#### Раздел "Цвета"
- Создание новых цветов (например: "Венге", "Орех")
- Указание HEX-кода для отображения

## Как это работает для дилеров

### Каталог на сайте
1. Дилер видит список серий с обложками
2. При клике — попадает на страницу серии
3. Видит все товары серии с фото по цветам
4. Может запросить прайс-лист через форму

### Фотографии
- Обложки серий — 800x600px или больше
- Фото товаров — ႂвадратные, 600x600px или 800x800px
- Поддерживаемые форматы: JPG, PNG

## API для интеграций

### Цвета
- GET /api/colors — список цветов
- POST /api/colors — создать цвет

### Серии
- GET /api/series — список серий
- POST /api/series — создать серию

### Товары
- GET /api/products-full — товары с вариантами и цветамов
- POST /api/products-full — создать товар с вариантами
